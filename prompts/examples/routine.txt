# 长期习惯 (Routine) 样例

## 创建习惯 [⭐⭐关键区别]
用户："我想每天健身"
→ **识别为 `create_routine`**
→ Default Frequency: "daily"
Execute:
```
[执行结果] 已创建长期习惯"每天健身"
- 建议时间: 20:00 (根据历史)
```

## 查询习惯 (Dashboard)
用户："查询今日习惯"
Execute:
```
[执行结果] 找到 2 个习惯待完成:
1. 每天健身 (20:00)
2. 每天背单词 (9:00)
```

## 修改习惯
用户："把健身改成每周三次"
Execute:
```
[执行结果] 已更新"健身"的频率
- 频率: 每周 (Weely)
- 指定日: 周一, 周三, 周五
```

## 删除习惯 (Protection Logic)
Case 1: 刚建立几天
-> **直接删除**
Execute:
```
[执行结果] 已删除"背单词"习惯
```

Case 2: 坚持超过 30 天且完成率 > 80%
-> **需确认**
Execute:
```
[需要确认] 您已坚持"健身" 45 天 (完成率 85%)，确定要放弃吗？
```

## 跳过单次
用户："今天不想健身了"
-> `mark_routine_completed` (Skipped status)
Execute:
```
[执行结果] 已跳过今天的"健身"
- 明天继续加油！
```
