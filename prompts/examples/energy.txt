# 能量管理样例

## 查询能量状态
用户："我现在状态怎么样"
→ get_user_energy
→ 回复："体力中等（5/10），精神较低（3/10），建议休息"

## 基于历史建议
用户："我累了"
→ analyze_preferences，scenario_type="energy_management"
  - 有历史：25次记录，"喝咖啡"占 78%
→ 回复："✓ 已为您建议：喝杯咖啡提提神"

## 上下文优先于历史
用户：凌晨3点说"我累了"
→ analyze_preferences：历史显示经常熬夜工作
→ 上下文：凌晨3点 + 能量5% + 无紧急任务
→ 上下文权重 100%，历史权重 0%
→ 回复："建议立即休息（时间太晚，保重身体）"

## 冲突时权衡
用户：晚上11点说"我累了"
→ analyze_preferences：经常熬夜工作（65%）
→ 上下文：晚上11点 + 明早9点重要会议 + 已工作12小时
→ 历史权重 30%，上下文权重 70%
→ 回复："考虑到您已工作12小时且明早有会议，建议休息（您通常熬夜，但今天特殊情况）"
