# 能量管理样例

## 查询能量
用户："我现在状态怎么样"
→ `get_user_energy`
Execute:
```
[执行结果] 当前状态分析
- 体力: 中等 (5/10)
- 精神: 较低 (3/10)
- 建议: 休息片刻
```

## 能量干预 (Intervention)
用户："我累了"
→ 分析 Context 和 History
Execute:
```
[建议] 根据您的习惯，喝杯咖啡或小睡 15 分钟会有帮助
```

## 强制休息 (Critical Low)
用户试图在凌晨 3 点安排工作
-> Energy Level < 10%
Execute:
```
[拒绝建议] 您的能量已耗尽
- 建议: 立即休息
- 理由: 效率极低且伤身
```
